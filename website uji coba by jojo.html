  1. agar para pembaca dapat tahu apa itu html dan fungsinya

1.apa itu html??

html merupakan singkatan dari Hyper Text Markup League,biasanya Html digunakan untuk membuat website atau pengembangan website

2.cara membuat kerangka dan short cut nya

untuk mulai membuat website kita perlu membuat kerangkanya terlebih dahulu
contoh:
    <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    </head>
    <body>
    </body>
    </html>

atau kalian bisa ketik"!" lalu klik "enter" untuk menggunakan short cut nya

3.gunakan pembuka dan penutup dari setiap tag,contohnya

  • <h1>halo dunia</h1>
  • <h2>halo semua</h2>

bisa dilihat,perbedaan nya ialah,dibagian penutup terdapat"/",dimana itu berfungsi untuk menutup tag

4.pengenalan nama menggunakan Html

    <!DOCTYPE html>
    <html lang="en">
    <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul kamu</title> <--------- ubah judul mu sesuai keinginan mu
    </head>
    <body>
    <h1>halo semua</h1> <---------tambahkan tag h1 sebagai awalan
    <h2>nama saya abi</h2> <---------tambahkan tag h2 sebagai pengenalan namamu </body>
    </html>

note:tag h1 adalah tag terbesar,tag h6 adalah tag terkecil,jadi semakin besar angka nya semakin kecil pula ukuran fontnya
